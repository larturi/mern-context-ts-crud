# MERN Stack con Context API (Subida de imagenes, TailwindCSS, Formik) 

1. Clonar proyecto

2. Instalar dependencias
```bash
npm install
```

3. Levantar la base de datos mongo

```bash
docker-compose up -d
```

4. Ejecutar el servidor

```bash
tsc
```

```bash
npm run dev
```
